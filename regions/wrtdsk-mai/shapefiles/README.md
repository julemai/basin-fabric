# Shapefiles

Create one subfolder for each basin. The folder name is the ID of the
basin, e.g. XYZ. Then place the shapefiles in those folder, i.e.
- XYZ_lp.shp
- XYZ_lp.dbf
- XYZ_lp.prj
- XYZ_lp.shx
- XYZ_lp.json

You can derive those shapefiles with steps 01-03 given you have a list
of basins.csv available. See example file for format.
